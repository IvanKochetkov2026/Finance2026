Настройка Firebase
Нужно один раз подключить базу данных.
Это бесплатно и займёт ~10 минут.
1Создай проект Firebase
Зайди на console.firebase.google.com → «Создать проект» → введи название → без Google Analytics → «Создать».
2Включи авторизацию
В меню слева: Authentication → «Начать» → вкладка «Sign-in method» → «Эл. адрес / пароль» → включи → «Сохранить».
3Создай базу данных
Firestore Database → «Создать базу данных» → «Начать в тестовом режиме» → выбери регион → «Готово».
4Получи конфиг
Шестерёнка (⚙️) → «Настройки проекта» → вкладка «Общие» → прокрути вниз → «Добавить приложение» → иконка </> → введи любое название → скопируй firebaseConfig.
5Вставь данные сюда
